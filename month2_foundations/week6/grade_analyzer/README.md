# Week 6 — Data Structures

## 📌 Description

This week focuses on understanding and using Python data structures such as lists, dictionaries, sets, and tuples. These are essential for organizing and managing data.

---

## 🧠 Concepts Learned

### Lists

* Creating lists
* Adding and removing items (`append`, `remove`, `pop`)
* Looping through lists

### Dictionaries

* Key-value pairs
* Adding and updating values
* Looping with `.items()`
* Accessing values using keys

### Sets

* Removing duplicates
* Creating unique collections

### Tuples

* Immutable data structures
* Accessing elements by index
