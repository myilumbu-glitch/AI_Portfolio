#Number Guessing Game

Description:
A command-line number guessing game where the computer chooses a random number between 1 and 10, and the player tries to guess it.
The game tracks the number of attempts and lets you play multiple rounds.

#Features:

Random number generation

Tracks attempts for each round

Hints: "Too high" / "Too low"

Play again option for multiple rounds

#How to Run:

Open number_guessing_game.py in your terminal or VSCode.

#Run the script:

python number_guessing_game.py

Enter your guesses following the prompts.

After guessing correctly, choose whether to play again or quit.
