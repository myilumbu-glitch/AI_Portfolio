## 1. Calculator

**Description:**  
A simple command-line calculator that can perform addition, subtraction, multiplication, and division.  
It allows continuous calculations and displays results for each operation.

**Features:**
- Handles `+`, `-`, `*`, `/` operations
- Continuous operation in a loop
- Validates input and handles errors
- Keeps a history of results

**How to Run:**
1. Open `calculator.py` in your terminal or VSCode.
2. Run the script:  
   ```bash
   python calculator.py
