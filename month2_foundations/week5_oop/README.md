# Week 5 — File Handling & JSON

## 📌 Description

This week focuses on working with files in Python. The goal was to understand how to read, write, and store data using files, including working with JSON format.

---

## 🧠 Concepts Learned

* Opening and closing files
* Reading from files
* Writing to files
* File modes (`r`, `w`)
* Using `with open()` (best practice)
* Introduction to JSON
* Serializing and deserializing data using `json`

---

## 📂 Files in this Week

* `main.py` → basic file reading and writing
* `journal.txt` → simple text file example
* `grades.txt` → file storing grade data
* `grade_analyzer.py` → calculates average grade using JSON

---

## 🚀 How to Run

Open terminal in this folder and run:

```bash
python main.py
```

or

```bash
python grade_analyzer.py
```

---

## 🧪 Example

### Input (grades.txt)

```json
[90, 85, 64, 72, 100]
```

### Output

```text
Average grade: 82.2
```

* How to process data after loading it from a file
