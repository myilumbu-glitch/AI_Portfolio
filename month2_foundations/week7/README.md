# Word Counter Project

## Description

This is a simple Python project that counts how many times each word appears in a text.

## Features

* Converts text to lowercase
* Removes punctuation
* Counts word frequency using dictionaries
* Saves results to a JSON file

## How to Run

1. Run the program:
   python main.py
2. Enter any text
3. View word counts in the terminal

## Example

Input:
Hello hello world!

Output:
hello: 2
world: 1

## Technologies Used

* Python
* JSON
